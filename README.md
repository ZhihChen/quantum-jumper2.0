# 量子跃迁者 - 游戏部署指南

这是一个基于HTML5 Canvas的2D平台跳跃游戏，玩家可以在不同维度间切换以解决关卡挑战。

## 部署到GitHub Pages的注意事项

### 音频问题解决方案

游戏在本地开发环境中一切正常，但部署到GitHub Pages可能会遇到背景音乐不播放的问题。已实施以下解决方案：

1. **改进的音频加载策略**：
   - 使用标准化的相对路径 (`./resources/`)
   - 设置 `crossOrigin = 'anonymous'` 解决CORS问题
   - 实现音频预加载机制

2. **增强的播放逻辑**：
   - 添加重试机制提高播放成功率
   - 优化错误处理和日志记录

3. **GitHub Pages特定建议**：
   - 确保音频文件路径大小写正确（GitHub Pages区分大小写）
   - 音频文件格式使用MP3以获得最佳兼容性
   - 对于背景音乐，游戏会在用户开始游戏时尝试播放，而不是自动播放

## 本地开发

1. 克隆仓库到本地
2. 进入项目目录
3. 运行Python服务器：
   ```bash
   python -m http.server 8000
   ```
4. 在浏览器中访问 https://zhihchen.github.io/quantum-jumper1.0/

## 游戏控制

- **方向键**：移动角色
- **空格键**：跳跃/快速切换维度
- **数字键1-4**：切换维度
- **鼠标滚轮**：循环切换维度
- **ESC键**：暂停/恢复游戏

## 游戏功能

- 4种不同维度，每种都有独特的物理特性
- 10个精心设计的关卡
- 收集量子碎片解锁下一关
- 背景音乐和音效增强游戏体验
- 响应式设计，适配不同屏幕尺寸

## 技术栈

- HTML5 Canvas
- JavaScript (原生)
- CSS3

## 资源文件夹结构

所有游戏资源都位于`resources`文件夹中：

- 背景音乐：background_ambient.mp3
- 音效：collect_shard.mp3, dimension_switch.mp3, player_jump.mp3, hazard_hit.mp3
- 图像资源：游戏角色、背景、UI元素等

## 常见问题排查

### 音频问题

1. **背景音乐不播放**：
   - 确保浏览器未静音
   - 尝试刷新页面并重新开始游戏
   - 检查控制台是否有音频加载错误

2. **音效正常但背景音乐不工作**：
   - 背景音乐文件大小较大，加载时间可能更长
   - 尝试在不同浏览器中测试

3. **GitHub Pages部署后问题**：
   - 检查资源路径是否正确（区分大小写）
   - 确认GitHub Pages已完全部署

### 其他问题

如果遇到游戏性能问题或其他错误，请尝试：
- 关闭浏览器扩展
- 清除浏览器缓存
- 更新到最新版本的浏览器
