# 量子跃迁者 - 项目概要

## 文件结构
```

├── instructions.html       # 游戏说明页面
├── main.js                # 游戏核心逻辑

│   ├── hero_quantum_explorer.png    # 英雄图像
│   ├── dimension_icons.png          # 维度图标
```

## 页面功能规划

### index.html - 游戏主页面
- **导航栏**：游戏标题、说明页面链接
- **游戏画布**：主要的游戏区域，使用Canvas实现
- **控制面板**：维度切换按钮、游戏状态显示
- **信息面板**：当前关卡、收集品数量、能量条
- **游戏菜单**：开始游戏、暂停、重新开始选项

### instructions.html - 游戏说明页面
- **游戏介绍**：量子跃迁者的背景故事
- **操作指南**：详细的键盘和鼠标控制说明
- **维度说明**：四个维度的特性和使用方法
- **游戏技巧**：进阶玩法和策略建议
- **返回游戏**：回到主页面的链接

## 技术实现

### 核心技术栈
- **HTML5 Canvas**：游戏渲染引擎
- **JavaScript ES6+**：游戏逻辑和交互
- **CSS3 + Tailwind**：界面样式和响应式设计
- **Web Audio API**：音效系统
- **RequestAnimationFrame**：游戏循环优化

### 游戏引擎特性
- **多维度物理系统**：每个维度独立的物理规则
- **粒子系统**：量子效果和视觉特效
- **碰撞检测**：精确的物体碰撞判定
- **状态管理**：游戏状态、关卡进度的统一管理
- **音效系统**：动态音效和背景音乐
